# Report Library

This folder is the repo-local source area for generated market and AISA strategy reports.

- Source of truth for editable Markdown and `.docx` artifacts: `reports/`
- Public mirrored copy for the deployed site: `public/reports/`
- Browser-friendly public index: `public/reports/index.html`

## ClawHub Reports

- `ClawHub_Plugin_Viral_Report_ZH.md` / `ClawHub_Plugin_Viral_Report_EN.md`
  What it is: ClawHub plugin breakout analysis, plugin-author patterns, and AISA retrofit opportunities.
- `ClawHub_10K_System_Report.md` / `ClawHub_10K_System_Report_ZH.md` / `.docx`
  What it is: The repeatable production-system report for ClawHub `10k+` skills and prolific authors.
- `ClawHub_10K_Boss_Brief_EN.md` / `ClawHub_10K_Boss_Brief_ZH.md` / `.docx`
  What it is: Short boss-facing brief extracted from the 10k system analysis.
- `ClawHub_Multi_Ranking_Report_ZH.md` / `.docx`
  What it is: Combined downloads / stars / installs ranking analysis and ranking-factor summary.
- `ClawHub_Viral_Boss_Report_ZH.md` / `.docx`
  What it is: Management summary for ClawHub viral-skill and viral-author strategy.
- `ClawHub_Top200_AISA_Conversion_Report_ZH.md` / `.docx`
  What it is: Top-200 merged opportunity queue showing which ClawHub skills are best suited for AISA conversion.
